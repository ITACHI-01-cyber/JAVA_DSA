# Java DSA Codebase

This repository is organized by folder, with each folder containing Java implementations for a specific category of data structures and algorithms.

## Folder overview

- `Array` - array-based problems and solutions.
- `LinkedList` - linked list implementations and exercises.
- `StackQueue` - stack and queue related code.
- `TreeGraph` - tree and graph algorithms.
- `SortingSearching` - sorting and searching algorithms.
- `Misc` - miscellaneous Java DSA programs and utilities.

## Using this repository

1. Navigate to the folder that matches the topic you want to study.
2. Open the Java files to review the implementation.
3. Run the code in your Java IDE or from the command line.

## Notes

- Each folder should contain files grouped by algorithm type.
- Add or update the README if you create new folders or reorganize code.
